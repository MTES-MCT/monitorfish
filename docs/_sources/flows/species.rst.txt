=======
Species
=======

The ``species`` flow extracts the list of species from 
`data.gouv.fr <https://www.data.gouv.fr/fr/datasets/liste-des-especes-a-but-detudes-statistiques-sur-la-peche/>`__.
and loads it to the ``species`` table of the Monitorfish database.

It is not scheduled to run and it only required to run when the list of species has been
manually updated on data.gouv.fr by the development team.
