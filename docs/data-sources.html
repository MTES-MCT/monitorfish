External data sources¶

VMS positions¶

VMS positions are received on an API endpoint provided by the Kotlin Springboot backend service.

ERS xml files¶

ERS raw xml files are ingested by the ers flow from the configured location where ERS xml files must be deposited.

Databases¶

Data is imported (and constantly updated) from external databases for Controls, Controllers, Fishing gears and Vessels.

These databases are :

  • OCAN

  • FMC2

Credentials for these data sources must be configured for Monitorfish to connect to them. See Environment variables.
